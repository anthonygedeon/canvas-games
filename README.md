<h1 align="center">Pong</h1>

## Installation

Install via pip:
```
pip install -r requirements.txt
```

## Play:
```sh
$ cd path/to/pong
$ python3 pong.py
```

## Controls

- UP: Up Arrow Key
- DOWN: Down Arrow Key
